# Grocery List App

This is a simple grocery list application built with Flask for the backend and React with TypeScript for the frontend.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Python 3.8+**
- **Node.js 14+** and **npm**

## Project Setup

### 1. Clone the Repository

```bash
cd grocery-list-app
```
